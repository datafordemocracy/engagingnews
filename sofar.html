So Far

What’s next?

The fun part!

  • extract common features from diverse sources
  • derived from theories of presidential/political accountability (e.g., complexity, tone/affect, keywords, topics, moral language, issue/policy attention, ideological position, named entities, and presidential events (e.g., agent-verb-object).)
  • represent features visually, dynamically, comparatively (openly, reproducibly)
  • experimentally validate how citizens “read” this information
  • iterate
  • plus, straight up social science on polarization, agenda-setting, etc. in media ecosystem
